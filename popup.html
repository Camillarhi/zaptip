⚡ zaptip

Tip GitHub profiles with Bitcoin Lightning.
  1. Open any GitHub profile that has a Lightning address in its bio (e.g. name@getalby.com).
  2. zaptip detects it and adds a ⚡ Zap button near the profile name.
  3. Click it, pick an amount (or enter a custom one), and confirm.
  4. If you have a WebLN wallet (like Alby), it pays directly. Otherwise a QR code appears to scan with any Lightning wallet.
No accounts, no settings, no servers. Payments go straight from your wallet to the recipient. Use a testnet/regtest address to try it safely.
